## Install ZooKeeper. 
 - Kafka needs zookeeper for cluster management.

## Install Kafka
 

## Start Zookeeper on windows
- cd <ZOOKEEPER_HOME>/bin
- zkServer.cmd

## Start Kafka on windows
- cd <KAFKA_HOME>/bin/windows/
- kafka-server-start.bat ../../config/server.properties
